# Topps F1 Price Tracker Dashboard

Live dashboard for monitoring Topps F1 trading card pricing and product launches.

## Features
- Full product catalog (2020-2025) with real-time pricing data
- Price tracker with Chrome, Sapphire, Dynasty historical charts
- Upcoming releases calendar
- 2020 Vintage Finds deep dive (Hobby, Sapphire & Dynasty)
- Personal watchlist for tracking deals

## Tech Stack
- React + Vite + Tailwind CSS + shadcn/ui
- Recharts for data visualization
- Static build deployed via GitHub Pages
